# Athletic Performance Analysis – University Rugby Team

This project presents an **Interactive PowerBI dashboard** and analysis of a university rugby team's performance.

## Project Highlights
- Player metrics: POWR (W), Relative Power (W/kg), Z-Score, Percentile  
- Position-based comparisons & trends  
- PowerBI dashboard with interactive charts
